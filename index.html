Git + GitHub, Jira & Slack

Beginner guide for modern software teams

Welcome

This site provides a beginner‑friendly introduction to three essential tools for collaborative software development: Git and GitHub for version control, Jira for project planning and tracking, and Slack for team communication. The tutorial summarizes key concepts, points you to high‑quality official documentation and videos, and includes hands‑on examples you can try on your own machine.

Contents

  • Git & GitHub – Learn how to create repositories, branch and merge using GitHub’s pull‑request workflow.
  • Jira – Get started with Jira spaces, boards, workflows and integrations.
  • Slack – Discover how to organize conversations and follow best practices for effective communication.
